## Design Elements

### Header
- Sticky on scroll
- Logo left, nav center, phone + CTA button right
- Mobile: hamburger menu, phone icon tappable
- Background: navy solid, white text, orange CTA button

### Hero
- Full-width background image (real project photo)
- Dark overlay (navy gradient, 60% opacity)
- Headline white, subtitle light gray
- CTA button: burnt orange, white text, rounded, shadow
- No stock photos — use real Huntsville fence projects

### Buttons
- Primary: Burnt Orange `#D4692C` bg, white text
- Hover: Darker orange `#B8571F`
- Secondary: White bg, navy border, navy text
- All: `border-radius: 6px`, `font-weight: 600`

### Cards (Services)
- White bg, soft shadow, rounded corners
- Icon top, title, short description, "Learn More →" link
- Hover: subtle lift (`translateY(-2px)`)
- 3-5 per row, responsive grid

### Testimonials
- White card, soft shadow
- Star rating top, quote text, name + location
- Carousel or slider for multiple entries

### Gallery
- Grid layout, 3 columns desktop, 2 tablet, 1 mobile
- Click to open lightbox
- Tabs/categories: All | Wood | Chain Link | Iron | Vinyl | Gates

### FAQ
- Accordion style, single open at a time
- Plus/minus icon toggle
- Navy question, dark gray answer

### Footer
- 3-4 columns: Logo/About, Quick Links, Services, Contact
- Dark navy background, white text
- Social icons (Facebook, Instagram, Google, Yelp)
- Copyright + legal links bottom
